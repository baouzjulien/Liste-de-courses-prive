lien de test de l'app : https://baouzjulien.github.io/Liste-de-courses/

Application simple de gestion d'une liste de courses avec classement par rayon "dragable", ajout d'articles dans les rayons et possibilité de cocher décocher les articles comme une checklist des courses en cours/réalisées. Les rayons comme les produits sont modifiables.

Ecrit en HTML/CSS et JS VANILLA avec persistence des données via localStorage
Installable depuis le lien sous forme d'une PWA (Progressive web application)
